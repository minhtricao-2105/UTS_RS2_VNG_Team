Callback function for receiving data from a R\+OS message. 

This function is called when a R\+OS message is received by the subscriber. The data from the message is stored in a global variable, and a pixel-\/to-\/local-\/frame transfer value is also calculated and stored in another global variable.

Callback function to receive color data from CV nodes.
