\doxysubsection{Detailed Description}
This file provides functions for detecting different colors in an image. 

It provides functions for detecting yellow, orange, and green colors, as well as detecting batteries in an image. These functions use Open\+CV to convert images to the H\+SV color space and apply thresholds to isolate the desired color. Contours are then found in the resulting mask to identify the object(s) of interest.
